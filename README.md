# Gerenciador de gastos

##  Problema
Muitas pessoas não controlam seus gastos e acabam com dificuldades financeiras.

##  Solução
Uma aplicação CLI simples para registrar e acompanhar gastos.

##  Funcionalidades
- Adicionar gastos
- Listar gastos
- Remover gastos
- Calcular total

##  Como executar
```bash
python src/main.py
